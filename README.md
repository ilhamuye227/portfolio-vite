# Portfolio (Vite + Vanilla JS)

Proyek ini adalah situs portofolio sederhana. Saya sudah mengonversinya agar bisa dijalankan dengan Vite (Vanilla JS).

## Perintah penting

- Install dependensi:

  npm install

- Jalankan dev server:

  npm run dev

- Build untuk produksi:

  npm run build

- Preview hasil build:

  npm run preview

Catatan: File `index.html` tetap di root; file JavaScript utama sekarang di `src/main.js`.
